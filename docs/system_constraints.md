# System Constraints

## Eye Tracking
- Dwell time: 1–1.5 seconds
- Blink confirmation: Optional

## Object Detection
- Refresh rate: Every 2 seconds
- Max objects passed to AI: 2

## AI
- Gemini response time target: < 2 seconds
- Max suggestions: 3

## UX
- Always provide touch fallback
- TTS must trigger in ≤ 1 tap
